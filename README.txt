The goal of this game is to fight against randomised monsters. A random monster will be selected each time you select the "Fight" option in the Main Menu. The three monsters available are the Slime, Zombie and Mutant Spider.

Created in my 1st year at university for the Programming Fundamentals module, where the brief was to create a simple text-based game in C++.

Controls
--------

Navigate through menus by typing 1-3 when prompted to do so.
